C++11 auf OS X

Published 

In Terminal

Install gcc4.7

brew tap SynthiNet/synthinet
brew install gcc47
ln -s /usr/local/bin/gcc-4.7 /usr/local/bin/gcc
ln -s /usr/local/bin/g++-4.7 /usr/local/bin/g++

Füge /usr/local/bin/ dem $PATH hinzu

nano ~/.profile



export PATH=/usr/local/bin/:$PATH

Lade PATH bei Eclipse Start

nano /Applications/eclipse-cdt/Eclipse.app/Contents/MacOS/eclipse.sh



#!/bin/sh
. ~/.profile
logger "`dirname \"$0\"`/eclipse"
exec "`dirname \"$0\"`/eclipse" $@

Executables Right für tiShell-Script

chmod +x /Applications/eclipse-cdt/Eclipse.app/Contents/MacOS/eclipse.sh

Change "Executable file" to eclipse.sh

open /Applications/eclipse-cdt/Eclipse.app/Contents/Info.plist

Regristriere neuen OS X Launcher beim System

/System/Library/Frameworks/CoreServices.framework/Frameworks/LaunchServices.framework/Support/lsregister -v -f /Applications/eclipse-cdt/Eclipse.app

In Eclipse:

Project Settings -> C/C++ Build -> Settings -> GCC C++ Compiler -> Micellaneous: -std=c++11
